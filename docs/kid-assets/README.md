# Kid Assets

This folder is where Anaya's and Kellen's contributions live. Drawings, voice recordings, written copy, naming lists.

## Structure

```
kid-assets/
├── anaya/
│   ├── oracle-design/         drawings, sketches, mood boards
│   ├── realm-names/           list of realm name ideas
│   ├── sidekick-names/        list of sidekick names
│   └── oracle-dialogue/       Oracle script lines
├── kellen/
│   ├── ui-style-direction/    Projects Tab visual ideas
│   ├── sound-preferences/     "saving a realm should sound like..."
│   ├── mode-names/            mode descriptions in his voice
│   └── ending-text/           destroy vs heal ending copy
└── shared/
    └── playtest-feedback/     written notes from playtest sessions
```

## How to Use

- Anaya and Kellen drag/drop their files here directly. No git knowledge required.
- Vanessa commits batches of kid assets at end-of-day, with credit in commit message.
- Claude Code reads from these folders during build to use real kid copy in the game.

## What Ships

Files in this folder may or may not ship in the final game depending on Phase scope. Even cut assets stay here as record of the kids' contribution.
